# Build

1. `git clone git@github.com:TrueBad0ur/cgit.git`
2. `git submodule update --init --recursive`
3. `cd cgit`
4. `make`
5. `sudo make install`

## Useful files
- everything to /var/www/htdocs/cgit/
- cgit(binary) -> usually renames to cgit.cgi
- cgit.js
- cgit.css
- cgit.png
- favicon.ico
- robots.txt
- filers/* to /usr/local/lib/cgit/filters/
